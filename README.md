# Deen & Discipline OS

A personal productivity and discipline system focused on Islamic lifestyle tracking.

## Features (MVP)

* Task management
* Prayer tracking
* Daily journal
* Discipline scoring system

## Tech Stack

* Django 5
* SQLite (development)

## Setup

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
